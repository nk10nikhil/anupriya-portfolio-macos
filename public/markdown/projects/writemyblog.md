# WriteMyBlog - Online Blogging Platform

A comprehensive blogging platform with WYSIWYG editor and SEO-friendly features.

## 🚀 Features

- ✍️ **WYSIWYG Editor**: Rich text editing experience
- 🔍 **SEO Optimized**: Meta tags and SEO-friendly URLs
- 📊 **Analytics Dashboard**: Track blog performance
- 🏷️ **Tagging System**: Organize content efficiently
- 📱 **Responsive**: Perfect reading experience on all devices
- 👤 **User Management**: Author profiles and permissions

## 💻 Tech Stack

- **Frontend**: Next.js, React
- **Backend**: Node.js
- **Database**: PostgreSQL
- **Editor**: Rich text WYSIWYG
- **SEO**: Next.js SEO optimization

## 🎯 Key Achievements

- Feature-rich WYSIWYG editor
- SEO-optimized for better visibility
- Comprehensive analytics dashboard
- Clean and intuitive user interface

## 📊 Key Features

- Create, edit, and publish blogs
- Rich text formatting
- Image upload and management
- Draft and published posts
- Category and tag management
- Author profiles
- Comment system
- Social sharing
- Search functionality
- Analytics and insights
