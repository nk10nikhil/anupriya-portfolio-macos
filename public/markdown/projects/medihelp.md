# MediHelp - AI Medical Chatbot

An AI-powered medical chatbot leveraging LLMs and LangChain for real-time conversational responses to medical queries.

## 🚀 Features

- 🤖 **AI-Powered Responses**: Utilizes advanced language models for accurate medical information
- 🔍 **Vector Search**: Pinecone-based vector search for efficient medical data processing
- 🌐 **Web Interface**: Flask-based web application for easy interaction
- 🐳 **Dockerized Deployment**: Containerized application for consistent deployment
- ☁️ **Cloud Infrastructure**: Deployed on AWS EC2 with ECR for container registry
- 🔄 **CI/CD Pipeline**: Automated deployment via GitHub Actions

## 💻 Tech Stack

- **Backend**: Python, Flask
- **AI/ML**: LangChain, OpenAI/Anthropic LLMs
- **Database**: Pinecone Vector DB
- **Cloud**: AWS (EC2, ECR)
- **DevOps**: Docker, GitHub Actions

## 🎯 Key Achievements

- Real-time conversational AI with medical knowledge base
- Efficient vector-based retrieval for fast response times
- Scalable cloud architecture with automated deployment
- Production-ready with Docker containerization
