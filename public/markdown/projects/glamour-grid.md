# Glamour Grid - Salon Branding Website

A stunning salon branding website built with accessibility and modern design principles.

## 🚀 Features

- ♿ **Accessibility First**: WCAG/ARIA standards compliance
- 🎨 **Vibrant UI**: Beautiful design with Tailwind CSS
- 📅 **Booking System**: Online appointment scheduling
- 🖼️ **Gallery Showcase**: Portfolio of salon work
- 📱 **Responsive**: Perfect on all screen sizes
- ⚡ **Performance**: Optimized for fast loading

## 💻 Tech Stack

- **Frontend**: Next.js, React
- **Styling**: Tailwind CSS
- **Architecture**: Component-based design
- **Accessibility**: WCAG 2.1 compliant

## 🎯 Key Achievements

- Full WCAG/ARIA accessibility compliance
- Modern, vibrant user interface
- Component-based architecture for maintainability
- Optimized performance metrics

## 📊 Key Features

- Service listings with pricing
- Online booking system
- Stylist profiles
- Before/after gallery
- Customer testimonials
- Contact form
- Location and hours
- Special offers section
