# Sudoku Solver

An intelligent Sudoku puzzle solver using backtracking algorithm with user authentication.

## 🚀 Features

- 🧩 **Backtracking Algorithm**: Efficient puzzle-solving algorithm
- 👤 **OAuth Authentication**: Google and Facebook login
- 💾 **User Data**: Save and track puzzle solutions
- 📱 **Responsive Design**: Works on all devices
- 🎯 **Interactive UI**: User-friendly puzzle interface
- 📊 **Solution History**: Track solved puzzles

## 💻 Tech Stack

- **Frontend**: Next.js, React
- **Backend**: Node.js
- **Database**: MongoDB
- **Authentication**: OAuth 2.0
- **Algorithm**: Backtracking

## 🎯 Key Achievements

- Efficient backtracking algorithm implementation
- OAuth integration with Google and Facebook
- User data persistence
- Clean and intuitive interface

## 📊 How It Works

The solver uses a backtracking algorithm to:

1. Find empty cells in the puzzle
2. Try numbers 1-9 in each cell
3. Check if the number is valid
4. Recursively solve remaining cells
5. Backtrack if no solution found
