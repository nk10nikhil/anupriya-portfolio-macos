# Bachat - Local MCP Expense Tracker

A local MCP server for personal expense tracking integrated with Claude Desktop for intelligent expense management.

## 🚀 Features

- 💰 **Expense Management**: Efficient CRUD operations on financial data
- 📊 **Smart Categorization**: Dynamic JSON-based category resources
- 🤖 **AI Integration**: Seamless Claude Desktop integration for interactive querying
- 📈 **Real-time Analysis**: Instant expense analysis and summarization
- 🔍 **Advanced Querying**: Natural language queries for expense insights
- 💾 **Local Storage**: SQLite-based storage for privacy and speed

## 💻 Tech Stack

- **Backend**: Python, FastMCP
- **Database**: SQLite
- **AI Integration**: Claude Desktop MCP Protocol
- **Storage**: Local JSON-based configuration

## 🎯 Key Achievements

- Seamless integration with Claude Desktop
- Natural language interface for expense management
- Fast and efficient local database operations
- Privacy-focused with local-only data storage
