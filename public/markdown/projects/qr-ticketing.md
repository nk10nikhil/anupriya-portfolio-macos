# QR-Based Ticketing System

A comprehensive QR-based ticketing platform built for SIH 2024 hackathon in just 36 hours.

## 🏆 Achievements

- 👥 **Managed 3,000+ attendees** with secure event registration
- ⚡ **30% improved check-in efficiency** with automated QR scanning
- 🚀 **90+ Lighthouse score** for performance optimization
- 📊 **Real-time dashboards** for event organizers
- ⏱️ **36-hour development** during SIH 2024 hackathon

## 🚀 Features

- 🎫 **QR Code Generation**: Unique QR codes for each attendee
- ✅ **Quick Check-in**: Fast and efficient attendee verification
- 📱 **Mobile Responsive**: Works seamlessly on all devices
- 📊 **Analytics Dashboard**: Real-time insights into event metrics
- 🔐 **Secure**: JWT-based authentication and authorization
- 📧 **Email Integration**: Automated ticket delivery

## 💻 Tech Stack

- **Frontend**: React, Next.js
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Methodology**: Agile Scrum
- **Version Control**: GitHub
