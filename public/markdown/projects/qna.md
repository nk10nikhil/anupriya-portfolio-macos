# Full-Stack Q&A System

A comprehensive Q&A platform with voting, reputation system, and community features.

## 🚀 Features

- ❓ **Post Questions**: Create detailed questions with formatting
- 💬 **Answer Questions**: Provide helpful answers
- 👍 **Voting System**: Upvote/downvote questions and answers
- 🏆 **Reputation System**: Earn points for contributions
- 🏅 **Badges**: Achievement system for active users
- 🔍 **Search**: Find questions quickly
- 🏷️ **Topics**: Follow topics of interest
- 👥 **Community Moderation**: Flag and moderate content

## 💻 Tech Stack

- **Frontend**: Next.js, React
- **Backend**: Node.js, Appwrite
- **Database**: Appwrite Database
- **Authentication**: Appwrite Auth
- **Search**: Full-text search

## 🎯 Key Achievements

- Complete Q&A platform functionality
- Reputation and badge system
- Community-driven moderation
- Advanced search capabilities

## 📊 Key Features
