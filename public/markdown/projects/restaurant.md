# Restaurant Booking & Branding Web App

A modern restaurant website with online booking system and Google Maps integration.

## 🚀 Features

- 🗺️ **Google Maps Integration**: Interactive location services
- 📅 **Real-time Booking**: Live table reservation system
- 📱 **Responsive Design**: Optimized for mobile and desktop
- 📈 **Business Impact**: 30% increase in online reservations
- 🎨 **Beautiful UI**: Modern and elegant design
- 📧 **Email Notifications**: Automated booking confirmations

## 💻 Tech Stack

- **Frontend**: React
- **APIs**: Google Maps API
- **Styling**: Modern CSS/Tailwind
- **Backend**: Node.js
- **Database**: MongoDB

## 🎯 Key Achievements

- 30% increase in online reservations
- Seamless booking experience
- Google Maps integration for easy navigation
- Mobile-optimized for on-the-go bookings

## 📊 Features Breakdown

- Online table reservation
- Menu display with images
- Location with map integration
- Contact information
- Gallery showcase
- Special offers and promotions
- Customer reviews
- Reservation management dashboard
