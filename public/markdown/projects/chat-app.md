# Real-Time Chat Application

A scalable real-time chat application supporting 500+ concurrent users with end-to-end encryption.

## 🚀 Features

- 💬 **Real-time Messaging**: WebSocket-based instant messaging
- 🔐 **End-to-End Encryption**: JWT-based secure authentication
- 👥 **Concurrent Users**: Supports 500+ simultaneous users
- 📱 **Responsive Design**: Works seamlessly on all devices
- 🧪 **Well Tested**: 95% test coverage with Jest & Cypress

## 💻 Tech Stack

- **Frontend**: React, Next.js
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Real-time**: WebSocket
- **Security**: JWT encryption
- **Testing**: Jest, Cypress
- **Deployment**: Vercel

## 🎯 Key Achievements

- Handles 500+ concurrent users
- 95% test coverage ensuring reliability
- End-to-end encryption for security
- Optimized for low-latency messaging

## 📊 Features Breakdown

- User authentication and authorization
- Private and group chat rooms
- Message history and persistence
- Online/offline status indicators
- Typing indicators
- File sharing capabilities
