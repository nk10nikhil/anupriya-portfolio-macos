# WeBuilt_U - Agency Website

A marketing agency website showcasing services and client work with stunning animations.

## 🚀 Features

- ✨ **Smooth Animations**: Framer Motion for engaging interactions
- 📈 **Business Impact**: 60% boost in user engagement
- 🎨 **Modern Design**: Built with React and Tailwind CSS
- 📱 **Responsive**: Optimized for all devices
- 🖼️ **Portfolio Showcase**: Client work and case studies
- ⚡ **Performance**: Fast loading and smooth scrolling

## 💻 Tech Stack

- **Frontend**: React
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Optimization**: Webpack, Lazy loading
