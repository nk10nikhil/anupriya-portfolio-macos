# SocioPulse - Social Media Platform

A feature-rich social media platform with real-time chat, media sharing, and community features.

## 🚀 Features

- 👥 **User Profiles**: Customizable user profiles with bio and avatar
- 💬 **Real-time Chat**: WebSocket-based instant messaging
- 📸 **Media Sharing**: Image and video upload capabilities
- ❤️ **Engagement**: Like, comment, and share posts
- 👤 **Social Network**: Follow/unfollow system
- 🔔 **Notifications**: Real-time activity notifications
- 📱 **Responsive**: Mobile-first design approach

## 💻 Tech Stack

- **Frontend**: Next.js, React
- **Backend**: Node.js, Express
- **Database**: Firebase
- **Real-time**: WebSocket
- **Storage**: Firebase Storage
- **Authentication**: Firebase Auth

## 🎯 Key Achievements

- Real-time messaging and notifications
- Scalable media storage solution
- Engaging user interface
- Community building features

## 📊 Platform Features

- Create and share posts
- Like, comment, and save content
- Follow users and build connections
- Real-time chat with friends
- Activity feed with personalized content
- User discovery and search
- Privacy settings and controls
