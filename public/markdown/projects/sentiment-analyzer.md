# Code-Mixed Sentiment Analyzer

Final year project implementing advanced NLP techniques for sentiment analysis on code-mixed text with graphemic normalization.

## 🚀 Features

- 🌐 **Code-Mixed Support**: Analyzes multilingual social media content
- 📝 **Graphemic Normalization**: Advanced text preprocessing for better accuracy
- 🎯 **High Accuracy**: Optimized machine learning models
- 📊 **Data Processing**: Efficient handling of normalized data
- 🔍 **Sentiment Classification**: Multi-class sentiment detection
- 📈 **Performance Metrics**: Detailed accuracy and F1-score reporting

## 💻 Tech Stack

- **Languages**: Python
- **Libraries**: NLTK, Scikit-learn, Pandas, NumPy
- **ML**: Machine Learning models for NLP
- **Processing**: Custom graphemic normalization algorithms
- **Visualization**: Matplotlib for results visualization

## 🎯 Key Achievements

- Successfully handles code-mixed Hindi-English text
- Advanced graphemic normalization for improved accuracy
- Optimized for social media content analysis
- Final year project demonstrating research capabilities

## 📊 Research Focus

This project addresses the challenging problem of sentiment analysis in code-mixed text, which is common in multilingual social media contexts. The graphemic normalization technique helps standardize the text before classification, leading to improved accuracy.
